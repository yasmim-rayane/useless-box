# Useless Box – Projeto Arduino

Este repositório contém o código em **Arduino (C++)** desenvolvido para o projeto **Useless Box**, um dispositivo interativo que reage a ações do usuário de forma imprevisível.

## Reconhecimento

O projeto foi **premiado com o primeiro lugar geral**, competindo contra todos os projetos de **Engenharia da Computação**.

## Objetivo do Projeto

O objetivo do projeto é demonstrar conceitos de **programação embarcada, controle de servomotores e interação homem-máquina**, criando uma experiência lúdica e inesperada a partir de um simples interruptor.

## Tecnologias Utilizadas

- Arduino
- Linguagem C++
- Servomotores
- Fita de LED RGB
- Módulo MP3 DFPlayer
- LEDs

## Funcionalidades

- Duas portas controladas por servomotores.
- Mecanismo interativo com um pequeno esqueleto.
- O esqueleto levanta e possui olhos brilhantes com LEDs.
- Fita de LED RGB com transição suave de cores em estado ocioso.
- Ação aleatória acionada por um interruptor.
